# AI-Assisted Customer Churn Analysis

## Overview
This project demonstrates an AI-assisted data analytics approach to analyze and predict customer churn for a subscription-based business.

## Tools Used
- Python (Pandas, NumPy, Scikit-learn)
- SQL
- Excel
- Power BI

## Key Work
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- AI-based churn prediction using Logistic Regression
- Dashboard creation and business insights

## Outcome
Identified high-risk customers and provided data-driven recommendations to support retention strategies.
